 =================== 版本 0.2 ===================
 
 状态             开发中

 最后修改日期     2009-11-9

 变更

 @ 增加文件最后访问日期

 @ 增加文件定位操作

 @ 改用文件描述符访问文件，向用户隐藏文件操作结构体
 

 =================== 版本 0.1 ===================

 状态             完成

 最后修改日期     2009-11-5

 支持的功能

 @ 支持文件的创建、打开、读写、定位、关闭、删除操作

 @ 可选的数据校验功能
 
 @ 可选的创建日期支持

 @ 自定义最大文件数
